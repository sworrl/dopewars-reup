Dope Wars: Re-Up

Install straight from your browser over USB — no tools to download.

  1. On the phone: Settings → About → tap Build number 7×, then enable USB debugging.
  2. Plug the phone into this computer with a USB cable.
  3. Click Connect below and approve the prompts (browser + the phone's "Allow USB debugging").
  4. Click Install.
Ready. Requires desktop Chrome or Edge (WebUSB).

⚠ This browser doesn't support WebUSB. Use desktop Chrome/Edge, or sideload manually with adb install.

Prefer the command line? Download the APK from Releases and run adb install dopewars-reup.apk.
